# Open Task Flow

Небольшой пайплайн, который периодически проверяет список задач и запускает выполнение найденной задачи.

## Что делает репозиторий

- Скрипт `pipeline.py` проверяет актуальные задачи и запускает выполнение.
- Запуск предполагается по расписанию через `cron`.

## Установка перед запуском

Перед использованием нужно:

- установить Python 3.12;
- установить `qwen cli`;
- установить `glab`;
- клонировать репозиторий с задачами (нужный для вашей среды) в директорию `target_repo/` этого проекта.

## Как добавить `pipeline.py` в cron

Пример добавления задания (запуск каждую минуту):

```bash
crontab -e
```

Добавьте строку:

```bash
* * * * * /usr/bin/python3.12 /Users/<YOUR_USER>/dev/open_task_flow/pipeline.py >> /Users/alex/dev/open_task_flow/cron.log 2>&1
```

При необходимости:

- замените путь к Python;
- замените путь к репозиторию;
- измените расписание.
